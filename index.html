Architecture

The system has brakes, not just answers.

01 Prompt / upload Local intake, no public legal-matter intake.
02 Route and classify Matter type, jurisdiction, support label, and risk posture.
03 Attach sources Official records, proof spans, manifests, and freshness checks.
04 Draft locally Prep-only output with deterministic validation.
05 Review lock Missing fact, Ask before draft, and human review required.
06 Production request File blocked, Research-only, or refused when unsafe.
